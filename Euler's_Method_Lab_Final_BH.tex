    \section{Euler's Method -
Introduction}\label{eulers-method---introduction}

The purpose of this exercise is to show how Euler's method can be used
to approximate the solution curves of differential eqautions. To
demonstarte this we will examine the trajectory of a projectile in two
cases:

\begin{itemize}
\tightlist
\item
  with NO air resistance
\item
  with air resistance
\end{itemize}

We have a differential equation, so we have the slope of the tangent
line at some initial point. If the curve is smooth, we use this slope to
find the enxt point, calculate the slope again to find the enxt point
and so on.

\(y = y_{0}+f\left(t_{0},y_{0} \right )\left(t-t_{0} \right ) \\ y_{n+1} = y_{n} +h f_{n}\)
